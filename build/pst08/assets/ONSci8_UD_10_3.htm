L’Encyclopédie canadienne : Le cycle hydrologique

Repère l’encadré « Ressources interactives » dans cette page Web. Clique sur l’hyperlien « Le cycle hydrologique ». Amuse-toi à explorer le cycle de l’eau à l’aide de cet outil interactif.

   

Curiosphere.tv : L’eau dans tous ses états !

Ce site Web interactif te propose des activités sur l’eau. Clique sur l’icône « Les changements d’état » et mets tes connaissances à l’épreuve ! Ensuite, fais de même pour l’icône « Le cycle de l’eau ».
   

CNRS : Cycle de l’eau et réservoirs – Les eaux souterraines
 

Lis ce bref article pour mieux connaître les eaux souterraines. Si tu le souhaites, clique sur la flèche au bas des pages pour te renseigner aussi sur les fleuves, les rivières et les lacs.
   

Environnement Canada : Les eaux souterraines – trésors cachés de la nature
 

Explore ce site Web pour connaître l’eau souterraine et son importance dans le cycle de l’eau. Ce site donne aussi de l’information sur les effets de la contamination de l’eau souterraine.
